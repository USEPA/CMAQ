CMAQ: The Community Multiscale Air Quality Modeling System#

This website contains documentation for the Community Multiscale Air Quality (CMAQ; “see-mak”) modeling system. Source code and sample scripts are provided in the CMAQ GitHub repository: USEPA/CMAQ

Note

This documentation is currently under development. Please email CMAQ_Team@epa.gov if you encounter any issues.

A brief introduction for new and returning users!
