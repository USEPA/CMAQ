CMAQ User’s Guide#

The CMAQ User’s Guide is designed to support the installation, configuration, and execution of the Community Multiscale Air Quality (CMAQ) model on Linux systems. CMAQ users should be comfortable with Linux scripting conventions and have some familiarity with the Fortran programming language. Users should also have some familiarity with atmospheric structure, and the physical and chemical processes that occur in the atmosphere.

Note: While this User’s Guide is packaged with the code when it is downloaded or cloned, users are encouraged to go online for the latest version.

Table of Contents#

Chapter 1 (Overview): CMAQ background, features, requirements and support resources.

Chapter 2 (Program Structure): Overview of the programs included with the CMAQ system.

Chapter 3 (Preparing Compute Environment): Hardware/software requirements for CMAQ default configuration.

Chapter 4 (Model Inputs): Preprocessing tools included with the base code and descriptions of the CMAQ input files.

Chapter 5 (Running CMAQ): Obtain the CMAQ source codes, set-up your environment, run a simulation.

Chapter 6 (Model Configuration Options): Configuration options for different CMAQ science modules.

Chapter 7 (Model Outputs): Descriptions of the CMAQ output files.

Chapter 8 (Analysis Tools): Post-processing, visualization and evaluation tools for CMAQ.

Chapter 9 (Process Analysis and Budget): Instrumented Models - Integrated Process Rates (IPR), Integrated Reaction Rates (IRR), and Budget Tool.

Chapter 10 (HDDM-3D): Instrumented Models - Decoupled Direct Method in Three Dimensions (DDM-3D).

Chapter 11 (ISAM): Instrumented Models - Integrated Source Apportionment Method (ISAM).

Chapter 12 (Sulfur Tracking): Instrumented Models - Sulfur Tracking Method (STM).

Chapter 13 (WRF-CMAQ): WRF-CMAQ model for simulating interactions between chemistry and weather.

Chapter 14 (MPAS-CMAQ): CMAQ is coulpled with the global Model for Prediction Across Scales (MPAS) allowing for seamless mesh refinement from global to local scales.

Tables and Figures: List of Tables and Figures.

Appendix A (Model Options): List of model options in configuration, compilation and run scripts.

Appendix B (Emissions Control): How to use the Emissions Control Namelist for customization of emissions processing.

Appendix C (Spatial Data): Information on how to create consistent geospatial data for CMAQ inputs.

Appendix D (Parallel Implementation): An introduction to how data-parallelism can be applied in the CMAQ system to increase computational efficiency.

Appendix E (Configuring WRF): Configuring the Weather Research and Forecasting Model (WRF) for use with CMAQ

Appendix F (ELMO Module): Description of the Explicit and Lumped CMAQ Model Output Module
