  • For rate constants with ko, ki, n, F values: k = [ koM/(1+koM/ki)]FG, where G=(1+(log10(koM/ki)/n)2))-1

  • For rate constants with k1, k2: k = k1 + k2M

  • For rate constants with k0, k2, k3: k = k0 + k3M/(1+k3M/k2)

  • For rate constants with k1, k2, k3: k = k1 + k2M + k3

  • For rate constants with the form A<Reference>, k equals A times a reference that represents photolysis rate, a heteorogeneous rate constant, rate constant for the given reaction or an operator. A equals one if not given.

  • In the mechanism definition file, the rate is formatted as

  • A~<HETEOROGENEOUS>

  • A*K<REACTION>

  • A/<PHOTOLYSIS>

  • A?<OPERATOR>
