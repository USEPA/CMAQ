CMAQ: The Community Multiscale Air Quality Modeling System
==========================================================

This website contains documentation for the [Community Multiscale Air Quality (CMAQ; "see-mak")](https://epa.gov/cmaq) modeling system. Source code and sample scripts are provided in the CMAQ GitHub repository: [https://github.com/USEPA/CMAQ](https://github.com/USEPA/CMAQ)

:::{note}
This documentation is currently under development. Please email <CMAQ_Team@epa.gov> if you encounter any issues.
:::
